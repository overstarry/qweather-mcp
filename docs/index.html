qweather-mcp

基于 MCP 的 QWeather 天气服务 CLI/服务器

功能亮点

  • 实时天气查询（温度、体感、风力、湿度等）
  • 多日天气预报（3/7/10/15/30 天）
  • 逐分钟降水、逐小时天气
  • 天气预警与生活指数
  • 空气质量（实时/小时/3日）
  • 可配置 API Base 与密钥

安装与运行

方式一：Smithery（推荐）

npx -y @smithery/cli install @overstarry/qweather-mcp --client claude

方式二：手动配置

  1. 在 QWeather 控制台获取 API Key
  2. 设置环境变量并启动：
export QWEATHER_API_BASE=https://api.qweather.com
export QWEATHER_API_KEY=<your-api-key>

# 启动 stdio 服务器
npx -y qweather-mcp

可用工具（部分）

  • get-weather-now（实时天气）
  • get-weather-forecast（多日预报）
  • get-minutely-precipitation（逐分钟降水）
  • get-hourly-forecast（逐小时预报）
  • get-weather-warning（天气预警）
  • get-weather-indices（生活指数）
  • get-air-quality / hourly / daily（空气质量）

开发提示

  • 构建：npm run build（TypeScript 编译并设置可执行权限）
  • Windows 构建：npm run build:windows
  • 源码：src/index.ts；编译输出：build/index.js
  • 需要环境变量：QWEATHER_API_BASE、QWEATHER_API_KEY
