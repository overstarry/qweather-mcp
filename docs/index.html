qweather-mcp

QWeather MCP server and CLI

Key Features

  • Real-time weather (temperature, feels-like, wind, humidity, etc.)
  • Multi-day forecasts (3/7/10/15/30 days)
  • Minutely precipitation and hourly forecast
  • Weather warnings and lifestyle indices
  • Air quality (real-time / hourly / 3-day)
  • Configurable API base and key

Install & Run

Method 1: Smithery (recommended)

npx -y @smithery/cli install @overstarry/qweather-mcp --client claude

Method 2: Manual setup

  1. Get your API key from the QWeather Console
  2. Export environment variables and start the server:
export QWEATHER_API_BASE=https://api.qweather.com
export QWEATHER_API_KEY=<your-api-key>

# start stdio server
npx -y qweather-mcp

Available Tools (selection)

  • get-weather-now (current conditions)
  • get-weather-forecast (3/7/10/15/30-day)
  • get-minutely-precipitation (next 2 hours)
  • get-hourly-forecast (24/72/168-hour)
  • get-weather-warning (alerts)
  • get-weather-indices (lifestyle indices)
  • get-air-quality / hourly / daily

Developer Notes

  • Build: npm run build (TypeScript compile + set executable bit)
  • Windows build: npm run build:windows
  • Source: src/index.ts; Output: build/index.js
  • Required env vars: QWEATHER_API_BASE, QWEATHER_API_KEY
